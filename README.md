# Godo

A to-do list app built with pure Go and HTMX, without any JavaScript.

## Development

### Prerequisites

You need to have following installed:

- Go
- Tailwind CSS
- Make

These binaries should be available in your `PATH`.


### Building

Simply run:

```
make
```

### Hot Reloading

To run the app with hot reloading, run:

```
make watch
```
